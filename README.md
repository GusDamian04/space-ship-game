## Juego Space Ship Game

El juego Space Ship Game basado en el jugeo Galaga

## Tecnología utilizadas

Las tecnologías utilizadas para este proyecto son:

- HTML5
- CSS
- JavaScript

Se utilizo canvas para manejar el área de juego
